# Parkinson's Disease Prediction using Machine Learning

## 📌 Project Overview
This project predicts Parkinson’s disease using biomedical voice measurements and supervised machine learning algorithms.

The model classifies patients into:
- Healthy (0)
- Parkinson’s Positive (1)

---

## 🧠 Problem Statement
Parkinson’s disease is a neurological disorder affecting movement. Early detection is critical for treatment planning.

This project builds a classification model to assist in early detection.

---

## 📊 Dataset Information
- Dataset: Parkinson’s dataset
- Features: Biomedical voice measurements
- Target variable: Status (0 = Healthy, 1 = Parkinson's)
- Total Features: 22+
